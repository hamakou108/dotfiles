# dotfiles

## Installation

Clone this repository.

```
$ git clone git@github.com:hamakou108/dotfiles.git
$ cd dotfiles
```

Run ``link.sh`` to create links to dotfiles.

```
$ ./bin/link.sh
```

Rename author name and email in `gitconfig`.

```ini:config/git/gitconfig
[user]
	name = <your_name_here>
	email = <your_email_here>
```
